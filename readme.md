## Project Introduce

**This is an experimental repo.**

This repo can write `export.css`, as soon as you saves `code.scss`.

## How to run

Run this project by using the following commands:

```bash
pnpm init           # init the pnpm
pnpm add -D sass    # install sass
pnpm exec sass --watch code.scss:export.css       # run the code.scss to export.css command
```
